### 这个项目能做什么
* 你可以搜索到QQ音乐的API和歌曲资源
* 你可以搜索到网易云音乐的API和歌曲资源
* 你可以搜索到我个人服务器上的歌曲资源,包括但不限于版权收费歌曲(我上传了多少你就能搜到多少)或是被封禁的歌曲(例如李志的歌曲)
### 技术栈/工具
* react:16.9.0
* antd:3.23.2

### 运行
`npm start`
